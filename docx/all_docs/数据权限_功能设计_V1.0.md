功能概述
概述
数据权限用于统一维护数据权限组及数据权限配置，规范系统内数据访问与使用范围。通过标准化数据权限控制规则，为不同角色和用户提供安全、可控的数据访问机制，保障系统数据的安全性和业务运行的合规性。
整体业务流程图
业务流程
用户使用场景
在系统使用前，由管理员为不同角色或用户配置相应的数据权限组和权限范围，完成配置后用户才能按授权访问相关数据；当组织职责或业务范围发生变化时，及时调整数据权限，确保数据访问安全、合规，避免因权限配置不当影响业务运行。
