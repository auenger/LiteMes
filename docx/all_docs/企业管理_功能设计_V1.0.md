功能概述
概述
企业管理用于统一维护公司、工厂、部门、班组、班制及日历方案等基础组织与时间数据，规范企业生产组织结构。通过标准化组织与作业时间配置，为生产计划、制造执行及质量管控提供统一的数据基础，保障业务运行的准确性和一致性。
整体业务流程图
业务流程
用户使用场景
在 MOM 系统运行前，由管理员统一维护公司、工厂、部门、班组及班制、日历等基础数据，完成组织与作业时间配置后相关业务才能开展；当组织或生产节奏调整时，及时更新基础数据，确保各业务模块运行准确，避免因基础信息错误影响生产执行。
