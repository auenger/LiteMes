功能概述
概述
设备主数据用于统一维护设备型号、设备类型及设备台账等基础设备信息，规范企业设备管理体系。通过标准化设备属性与分类配置，为设备运行管理、生产调度、维护保养及质量追溯提供统一的数据基础，保障业务运行的准确性和一致性。
整体业务流程图
业务流程
用户使用场景
在设备投入生产或系统启用前，由相关人员统一维护设备型号、类型及台账等基础信息，完成配置后设备才能参与生产与管理；当设备状态或属性发生变化时，及时更新主数据，确保生产执行、设备管理及相关业务运行准确，避免因设备信息不一致影响生产。
