功能概述
概述
物料主数据用于统一维护单位、单位换算比例、物料分类、物料、物料有效期及免检清单等基础物料信息，规范企业物料管理体系。通过标准化物料属性与计量规则配置，为生产计划、物料领用、库存管理及质量管控提供统一的数据基础，保障业务运行的准确性和一致性。
整体业务流程图
业务流程
用户使用场景
在业务开展前，由相关人员统一维护物料及其单位、换算关系、有效期和免检规则等基础信息，完成配置后物料才能被正确使用；当物料属性或管理要求发生变化时，及时调整主数据，确保生产、库存及质量业务执行准确，避免因物料信息错误影响生产和管控。
