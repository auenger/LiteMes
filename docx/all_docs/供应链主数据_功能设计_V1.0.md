功能概述
概述
供应链用于统一维护客户和供应商等基础往来单位信息，规范企业内外部业务关系管理。通过标准化客户与供应商数据配置，为订单管理、物料采购、生产协同及质量追溯提供统一的数据基础，保障业务运行的准确性和一致性。
整体业务流程图
业务流程
用户使用场景
在业务开展前，由相关人员统一维护客户和供应商等基础信息，完成配置后订单、采购及相关业务才能正常执行；当客户或供应商信息发生变化时，及时更新数据，确保供应链相关业务运行准确，避免因基础信息不一致影响业务协同。
