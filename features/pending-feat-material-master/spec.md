## 需求描述
管理 PCB 制造所需的物料基础数据，包括计量单位、单位换算、物料分类（多级树形）、物料基本信息（编码/名称/规格/分类/版本）及免检清单。

### 子功能
| 子功能 | 核心实体 | 关键规则 |
|--------|---------|---------|
| 计量单位管理 | `Uom` | 编码与名称唯一，支持计算精度设置 |
| 单位换算比例 | `UomConversion` | 原单位↔目标单位换算率，支持多级换算 |
| 物料分类 | `MaterialCategory` | 多级分类树形结构 |
| 物料基本信息 | `MaterialMaster` | 编码/名称/规格/分类/版本，版本管理支持 A.1→A.2 更迭 |
| 免检清单 | `InspectionExemption` | 供应商+物料+有效时间范围定义免检规则 |

## 用户价值点
1. 统一管理计量单位与换算，确保物料数据精度一致
2. 多级物料分类体系，支持 PCB 原材料/半成品/成品的分类管理
3. 物料版本管理，追踪物料规格变更历史
4. 免检规则配置，减少不必要的来料检验流程

## 验收标准 (Acceptance Criteria)

### 用户故事
作为物料管理员，我希望管理物料主数据，以便为采购、生产、质量追溯提供标准化的物料基础信息。

### Gherkin 验收场景

#### 场景 1: 物料编码唯一性
```gherkin
Given 物料编码 "MAT-001" 已存在
When 再次创建编码为 "MAT-001" 的物料
Then 返回错误 "物料编码已存在"
```

#### 场景 2: 物料版本管理
```gherkin
Given 物料 "MAT-001" 当前版本为 A.1
When 创建新版本 A.2
Then 新版本正确关联到同一物料
And 编码创建后不可修改
```

#### 场景 3: 免检清单
```gherkin
Given 供应商 S001 和物料 MAT-001 已存在
When 创建免检规则（供应商 S001 + 物料 MAT-001 + 有效期）
Then 免检规则生效
And 过期后自动失效
```
